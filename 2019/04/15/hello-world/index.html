重庆之旅
发布于 2019-04-15 / 更新于 2019-04-15 / 和宝贝的旅游 / 热度 0 / 喜欢 0

出发前

出发前一天，我们一起去吃了心心念念的海底捞火锅。幸好有学生证，不然还是挺贵的。
虽然吃着海底捞，心里想的却是明天和你的重庆之旅。

这里送宝贝一张海底捞使用券，下载保存图片即可

Day1

急急忙忙的上了动车，来到重庆已经中午了，我们已经饿得不行了，第一站就是解放碑的小吃！


好看的宝贝！

吃饱喝足，开始逛解放碑！实话说，解放碑没春熙路好。

累了，开始休息，这两张是我最喜欢的两张照片

走累了，我们买了点水果，回酒店休息了一会儿，我睡得很香嘿嘿。
来到重庆一定要去吃这里的火锅，还不错！

这里的火锅是干辣，不是很麻。很适合我们两个的口味

Day2

我们去逛了磁器口，人实在是太多，没一会儿就被挤的不行了，只好打道回府。我们还去了重庆的天街，感觉天街对于我们两个来说，还是挺有缘分的哈哈哈哈
对于我们两个，还有缘分的就是猫爪烤肉，所以来到重庆我们去吃了当地的猫爪烤肉，但是感觉这里的服务和成都比真的是两边天

重庆虽然到处都不是很适合我们，但是他的夜景是真的美丽，晚上的洪崖洞


Day3

第三天我们逛了很多地方，去了很多nike店，感谢小宝贝的陪伴！你穿小裙子也是最漂亮的

感谢小宝贝一路陪我！喜欢和你旅行，以后我们还要去更多更远的地方。希望你能一直陪着我！
